使用說明

Current State 頁面

點選 cell 旁的 icon 即可在頁面右側顯示該 cell 中的 device 狀態。

History State 頁面

透過上方的 Cell ID, IoT ID, 時間 可以選擇需要的歷史紀錄，Cell ID 與 IoT ID 不選擇為所有 Cell ID 與 IoT ID。查詢完後可以點選右方的 download 即可下載該 csv 檔案。

System Log 頁面

點選 DNS Call Flow 或 System Log 可選擇欲觀看之 log。並可透過時間來過濾 log 的區間。查詢完後可以點選右方的 download 即可下載該 csv 檔案。
